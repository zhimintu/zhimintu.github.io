TruckSim8.1与Matlab2018a联合仿真找不到TruckSim S-Function的解决办法

Posted on 2020-06-07 Edited on 2020-06-09

划重点：F5 或者 刷新Simulink Library Browser

1、在TruckSim中点击Send to Simulink

2、待Simulink打开之后，点击Simulink Library Browser

3、出现Simulink Library Browser 如果没有出现TruckSim S-Funciton也没有出现提醒信息“Fix”；则右键任意条目点击Refresh刷新或者F5；出现提醒信息“Fix”则直接点击Fix

4、最终出现了TruckSim S-Function。（掌声）
